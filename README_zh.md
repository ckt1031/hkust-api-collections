# HKUST API

[English](README.md)

這是針對香港科技大學 (HKUST) 線上服務的爬取 API。

這些 API **不是香港科技大學官方提供的**，而是通過逆向工程獲得的，可能會在沒有通知的情況下更改。

我們**不**提供獲取 ACCESS TOKEN 的公開方法，你需要自己找到獲取它的方法。
